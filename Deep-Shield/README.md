# DeepShield

DeepShield is a hackathon-ready web application for digital safety education. It provides:

- **Deepfake Detector**: Upload an image and estimate the likelihood that it is AI-generated.
- **Headline Scanner**: Paste a news headline and analyze it for emotional tone, sensationalism, and manipulation risk.

This project is split into two apps:

- `deep-shield-backend`: FastAPI + Transformers service on `http://localhost:8000`
- `deep-shield-frontend`: Next.js (App Router) UI on `http://localhost:3000`

---

## Backend – FastAPI (deep-shield-backend)

### Tech

- FastAPI
- Uvicorn
- Transformers (HuggingFace)
- Torch
- Pillow
- Python-multipart

### Endpoints

- **POST `/analyze-image`**
  - Input: multipart image file under field name `file`
  - Uses a pretrained vision transformer (`google/vit-base-patch16-224`) to estimate how likely the image is AI-generated by treating model uncertainty as a proxy.
  - Response:
    ```json
    {
      "ai_probability": 0.82,
      "confidence_level": "High",
      "explanation": "string"
    }
    ```

- **POST `/analyze-headline`**
  - Input JSON:
    ```json
    { "headline": "BREAKING: You won't believe..." }
    ```
  - Uses a pretrained text classification model (`distilbert-base-uncased-finetuned-sst-2-english`) plus keyword heuristics.
  - Response:
    ```json
    {
      "manipulation_score": 0.76,
      "risk_level": "High",
      "flagged_phrases": ["BREAKING", "YOU WON'T BELIEVE"],
      "summary": "string"
    }
    ```

### Backend Setup

From the project root:

```bash
cd deep-shield-backend
python -m venv .venv
source .venv/bin/activate  # Windows PowerShell: .venv\Scripts\Activate.ps1
pip install --upgrade pip
pip install -r requirements.txt
```

### Run Backend

```bash
cd deep-shield-backend
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

The API will be available at `http://localhost:8000` and docs at `http://localhost:8000/docs`.

## Frontend – Next.js (deep-shield-frontend)

### Tech

- Next.js (App Router, TypeScript)
- Tailwind CSS
- Axios
- Chart.js + react-chartjs-2

### Frontend Setup

From the project root:

```bash
cd deep-shield-frontend
npm install
```

### Run Frontend

```bash
cd deep-shield-frontend
npm run dev
```

Then open `http://localhost:3000` in your browser.

---

## Demo Flow

1. **Start backend**: `uvicorn main:app --reload` from `deep-shield-backend`.
2. **Start frontend**: `npm run dev` from `deep-shield-frontend`.
3. Navigate to `http://localhost:3000`.
4. Use the tabs to switch between:
   - **Deepfake Detector**:
     - Drag-and-drop or upload an image.
     - View AI probability, confidence badge, bar chart, and explanation text.
   - **Headline Scanner**:
     - Paste a headline.
     - View manipulation score, risk badge, highlighted buzzwords, and summary explanation.

Both tools are designed for **live demos** in classrooms, workshops, and hackathons. Treat all outputs as assistive signals, not definitive judgments.
